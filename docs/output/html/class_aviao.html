Goes through the list of flights and if there is a flight with the same parameters as the ones given then the flight is performed. It also adds the flight done as well as any other flight with a previous date to the vector of completed flights It lets the user know the specifics of the Car that transports their bag should they choose the auto check in option Finally it rewrites the list of flights so that we are only left with the ones still yet to be done
