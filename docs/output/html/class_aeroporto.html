Changes the desired plane to have the same attributes as the given plane, used when the flight is made and the it is needed to save the changes made to the plane's services and flight list. It also goes through every plane and changes their flights so that every flight that happens on a date before is considered as completed, because if you fly on a certain date, it wouldn't make sense to still be able to book a flight to a previous date It does the same to the services and in the end every plane is synchronized in terms of the date of their available flights and services
